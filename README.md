# Mapa Electroindustrial de México

Mapa interactivo de 45 proyectos de inversión en la cadena electroindustrial mexicana (~$30B USD anunciados), con cuatro capas de datos públicos para evaluar capacidades estatales.

**[Ver mapa](https://gilix.github.io/mexico-electroindustrial-map/)**

## Capas de datos

- **Afinidad productiva** — Ventaja comparativa revelada (RCA) combinando establecimientos DENUE y comercio exterior HS4
- **Capital humano** — Empleados registrados en DENUE 2022 por códigos SCIAN relevantes a cada categoría
- **Egresados** — Pipeline de talento universitario (matrícula ANUIES) por carrera y estado
- **Energía renovable** — MW instalados de eólica y solar fotovoltaica (PRODESEN 2023)

## Categorías

Baterías, Centros de datos, Electromovilidad, Generación solar, Manufactura solar, Manufactura VE, Semiconductores, Transmisión y distribución.

## Fuentes

- [DataMexico](https://datamexico.org/) — DENUE, comercio exterior HS4, ANUIES
- [PRODESEN 2023](https://www.gob.mx/sener/documentos/prodesen-2023-2037) — Capacidad instalada renovable por estado
